# Gentlman20211010

해당 제작물의 모든 소유권(소스,정보,파일,개발과정 등)은 (주)프시케가 갖는다.
(주)프시케에서 지정한 소속 관리자 외 일체의 열람/복사/사용을 금한다.
무단 열람/복사/사용이 되는 경우 해당 개인/단체에 법적 책임을 물을 수 있다.
이 외 상세사항 및 권한변경은 (주)프시케의 내부규정에 의거한다.

Gentlman20211010는 주로 Gentlman V6.3을 개발/관리를 위해 제작되었다.
Gentlman V6.3의 주 개발 목표는 다음과 같다.
 - V6.2를 베이스로 한 V6.3 PCB 설계 및 아트웤
 - 무료 툴인 KiCAD를 이용, 툴의 숙달 및 결과물 제작
 - V6.3에 맞는 펌웨어 제작
 - 블루투스 버전인 V6.3B의 개발
